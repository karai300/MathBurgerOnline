Generalized Burger Function

At the heart of Math Burger lies the Generalized Burger Function. Each layer of the burger can be expressed using this function.
Here the arguments of the function (a, b, w, v, r0) are parameters, and θ and φ are the variables which range 0 to π and 0 to 2π, respectively. The returned 3-element vector represents a point in three-dimensional space expressed by cylindrical coordinates.
Cylindrical coordinates
The triplet (r, φ, z) corresponds to the radius r, the azimuth angle φ, and the vertical height z, respectively. The cylindrical coordinates best describe axisymmetric shapes, like most layers of Math Burger.
Sphere in the cylindrical coordinates
Physicists love spheres, because it is the most basic shape. A sphere with a unit radius in the cylindrical coordinates can be represented by S(θ, φ)=(sinθ, φ, cosθ)  (0<θ<π, 0<φ<2π).
S(θ, 0)=(sinθ, 0, cosθ)  (0<θ<2π) represents a unit circle on φ=0 plane. And, φ rotates the circle around the z axis.
Deforming a sphere
We want to deform this sphere to make it into the shape of a bun, an onion, and so on.
If we think about the above function of the sphere, it can be written as:
S(θ, φ)=(sinθ, φ, f(cosθ)),
where f(x)=x.
If we replace this f(x) from the linear function x, the sphere is going to be deformed. As we use x=cosθ, f(x) is required to be defined for x between -1 and 1.
Here, we introduce the Gauss error function Erf(x). This function is asymptotically linear near the origin, but it starts to saturate at about |x|>0.5, and it asymptotes to +1 (or -1) as |x| becomes large.
If we go back to the sphere function with f(x)=Erf(x), this has the z value not as large as expected as |x| increases. In other words, the sphere is vertically squeezed.
Vertical asymmetry
Introducing a parameter a as f(x) = a x i.e. z=Erf(a cosθ), we can control how early the error function saturates. The figure below shows S(θ, φ a)=(sinθ, φ, Erf(a cosθ)) with the parameter a swept as 0.1, 0.5, 1, 3 (from left to right). If a is small, f(x)=Erf(a x) behave close to a linear function f(x)=ax, and therefore the sphere is uniformly squeezed in the vertical direction. When a is large, f(x)=Erf(a x) saturates to +/-1 very quickly, so the shape becomes like a cylinder.
Furthermore, using another parameter b as f(x) = Erf[a (x - b)], the vertical asymmetry can also be introduced. Below, a is fixed at 2 while b is swept as 1, 0.5, 0.
Translation and scaling
Now the things are much more simple. Let's introducing two more parameters, v and w for vertical translation and uniform scaling, respectively. i.e. f(x) = v+w Erf[a (x - b)]. This is the z component of the generalized burger function!
Wait!? What's happen to the integral in the burger function shown at the beginning? No worry! The error function Erf(x) is defined by the integral of the famous Gaussian function.
Last thing: add one more parameter r0 to the r component, to scale the horizontal size of the shape. Finally, we get the generalized burger function. Here the variables θ and φ are not shown in the argument, and the range of the variables are 0<θ<π and 0<φ<2π.

What to do with cheese and lettuce

The cheese and lettuce differ from the other parts in several ways:
1) The shape is rather 2D (or 2D like) than 3D.
2) The radial shape is not axisymmetric, but more like n-fold symmetric.
3) The shape also has vertically deformation, which is m-fold symmetric. (n ≠ m acceptable)
Compactification of an extradimension
A cylinder is compressed vertically to create a pseudo 2-dimensional disk. The below figure shows B[θ, φ, 3, 0, w, 0.46, 0.98] with w=0.5, 0.2, 0.05 from left to right.

Fourier series expansion of radial functions
How do we deal with the non-axisymmetric shape?

A few years before the French Revolution, Marie Antoinette said to a function-starved citizen, "If it's not axisymmetric, let them use Fourier series."
Exactly... In the Burger function, the radial coordinate is represented by r0 Sinθ, so we can just manipulate r0 as a function of the azimuthal angle φ. And, any function can be expressed by the Fourier series...
Below:
Left: B[θ, φ, 3, 0, 0.05, 0.46, 0.98]
Right: B[θ, φ, 3, 0, 0.05, 0.46, (0.98 + 0.05 sin4φ - 0.01 cos8φ)^2]
Below:
Left: B[θ, φ, 3, 0, 0.01, 0.14, 1]
Right: B[θ, φ, 3, 0, 0.01, 0.14, 1 + 0.1 sin6φ]
Fourier series expansion of the vertical deformation
But... how do you incorporate dripping cheese gravity and ripple of lettuce waves into the theory?
...Aha! The parameter v is not only for object translation. Any kind of vertical deformation can be expressed by modifying the parameter v as a function. In addition, the gravity can be expressed by synchronized vertical deformation to the radial function (previous section), while waves can be expressed by asynchronous dependence.
Here (sinθ)^16 is the key to make the experiment and the theory matches the best.
Cheese:
B[θ, φ, 3, 0, 0.05, 0.46 - 0.004 (1 + sin4φ)^6 (sinθ)^16, (0.98 + 0.05 sin4φ - 0.01 cos8φ)^2]
Lettuce:
B[θ, φ, 3, 0, 0.01, 0.14 + 0.05 cos32φ (sinθ)^16, 1 + 0.1 sin6φ]
